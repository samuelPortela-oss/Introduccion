Este es el sitio web para PokerMind, una plataforma de aprendizaje de idiomas a través de la técnica de repetición especiada. 
En la página web las personas podrán (una vez implimentadas todas las herramientas para tal efecto) descargar la aplicación 
de escritorio o celuar. En Java se realizará validación del dispositivo que usa la personas, a través de las cookies, para mostrarle 
la opción de escritorio o movil. 
También podrán registrarse o ingresar con sus usuarios y contraseñas. La información se envía al backend, que validará en Java, que 
el formato de correo esté correcto y la contraseña cumpla los parametros requeridos de comnbinar números y letras. 
En el caso de registro, validará los datos y anexará la información en la base de datos. Para el ingreso de usuario, validará 
la información y permitirá la entrada en caso de ser verdaderos o pedirá volver a ingresar los datos en caso de falso. 
Tendrá un contador de repeticiones que pedirá cambiar la contraseña después del 3 intento fallido.
